Glosario de Tecnología de Espectáculos

Control y Comunicación

MIDI (Interfaz Digital de Instrumentos Musicales)

Qué es: Protocolo de comunicación entre instrumentos musicales y software.

Para qué sirve: Enviar notas, controles (volumen, efectos) y sincronizar dispositivos.

Qué hace: Permite grabar, editar y reproducir música digital (ej.: conectar un teclado a una computadora).

DMX (Digital Multiplex)

Qué es: Protocolo estándar para controlar equipos de iluminación (luces robóticas, dimmers, etc.).

Para qué sirve: Enviar señales desde una consola o software para sincronizar espectáculos de luces.

Qué hace: Controla intensidad, color, movimiento y efectos de las luces en tiempo real.

OSC (Open Sound Control)

Qué es: Protocolo alternativo a MIDI, más flexible y moderno.

Para qué sirve: Comunicar dispositivos y software (arte interactivo, instalaciones, música).

Qué hace: Transmite datos complejos (coordenadas 3D, sensores) en redes locales.

Art-Net/sACN

Protocolos de red DMX sobre Ethernet.

SMPTE

Código de tiempo para sincronización A/V/Iluminación.

Timecode

Sincronización temporal.

Conexiones Físicas

XLR

Conector balanceado de audio/DMX.

TRS/TS

Conectores de audio balanceado/no balanceado.

Ethernet

Conector de red para datos/audio digital/control.

Speakon/PowerCON

Conectores de bloqueo para altavoces/alimentación.

BNC

Conector de bloqueo para video/sincronización.

SDI

Serial Digital Interface (video digital profesional).

HDMI

High-Definition Multimedia Interface (video/audio digital).

Software

DAW

Qué es: Software para hacer música (grabar, editar, mezclar).

Para qué sirve: Producir canciones, beats o sonidos en computadora.

Qué hace: Permite grabar voces/instrumentos, usar instrumentos virtuales (VST) y ordenar pistas.

Ejemplos: Ableton Live, FL Studio, Pro Tools, Logic.

Ableton Live

Qué es: DAW (Digital Audio Workstation) para producción musical y presentaciones en vivo.

Para qué sirve: Producir música, mezclar pistas y tocar en directo con loops.

Qué hace: Secuenciar audio/MIDI, automatizar efectos y sincronizar hardware.

Lighting Console Software

Software de control de iluminación.

Media Server Software

Software de gestión/reproducción de video.

Show Control Software

Software de integración/automatización de sistemas.

Resolume

Qué es: Software de VJing y mapping proyectivo.

Para qué sirve: Manipular y mezclar videos en vivo (conciertos, instalaciones).

Qué hace: Superponer capas, aplicar efectos y sincronizar con audio o DMX.

Blender

Qué es: Software gratuito de modelado 3D, animación y renderizado.

Para qué sirve: Crear gráficos 3D, efectos visuales (VFX), videojuegos o motion graphics.

Qué hace: Herramientas para esculpir, texturizar, simular física y editar video.

VVVV (o 4v)

Qué es: Entorno de programación visual para tiempo real (arte generativo, instalaciones interactivas).

Para qué sirve: Crear visuals, controlar hardware (DMX, OSC), procesar video/sensores.

Qué hace: Conectar nodos (sin código) para generar gráficos, sonido o interactividad.

Ejemplo: Controlar proyecciones con un Kinect + OSC + luces DMX.

Projector

Dispositivo de proyección de video.

Proyector (Tiro Corto)

Proyector (Láser)

Utiliza fuente de luz láser.

LED Screen

Pantalla de video modular LED.

Conmutador de fuentes de video.

Scaler

Términos de Video

Señal de Video Analógica

Señal de video continua.

Señal de Video Digital

Señal de video discreta (bits).

Resolución de Video

Número de píxeles en la imagen.

Tasa de Refresco

Cuadros por segundo (fps) o Hertz (Hz).

Términos Generales

Patching

Conexión física y asignación virtual.

Routing

Dirección de señales entre equipos.

Rigging

Estructura de soporte para equipos.

Cue

Qué es: Marca que indica un momento específico para ejecutar una acción.

Para qué sirve: Sincronizar diferentes elementos del espectáculo.

Qué hace: Permite activar luces, sonido o video en momentos precisos del show.

Playback

Qué es: Sistema de reproducción de contenido pregrabado.

Para qué sirve: Ejecutar audio, video o secuencias de luces durante un espectáculo.

Qué hace: Permite reproducir pistas de música, videos o secuencias programadas desde un dispositivo.

Live Feed

Qué es: Transmisión de video o audio en tiempo real.

Para qué sirve: Mostrar imágenes captadas en vivo durante un evento.

Qué hace: Captura y envía señal desde cámaras a pantallas o sistemas de streaming sin demora.
